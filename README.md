# 🌍 Code Journey

**Code Journey** is an interactive learning platform designed to teach **web development** concepts — starting from **HTML, CSS, and JavaScript**, and gradually extending to **Databases and Backend technologies**.

The project uses a modern front-end stack with **React + Vite**, combining **speed, scalability, and interactivity**. It’s crafted to make learning **visual, animated, and hands-on** — helping users not just read code, but actually **see it come alive**.

🔗 **Live Project:** [https://codejourney.space](https://code-journey-zeta.vercel.app/)

---

## 🧭 Project Purpose

The idea behind Code Journey is simple:
> *Make learning code more intuitive and engaging.*

Most beginner tutorials focus on text — but web development is **visual** by nature.  
So Code Journey:
- Shows **how code affects the page visually.**
- Provides **interactive examples** users can play with.
- Organizes topics into **clear, modular pages** for each language or tool.
- Uses **modern design** so learning feels like exploring, not reading a textbook.

---

## 🧱 Tech Stack Breakdown

### 🖥️ Frontend
- **React 18** — Dynamic component-driven UI.
- **Vite** — Ultra-fast build and hot reload system.
- **Tailwind CSS** — Design consistency and rapid UI building.
- **Framer Motion** — Smooth animations and transitions.
- **Shadcn/UI** — Pre-styled modern React components.
- **Lucide Icons** — Crisp, minimal iconography.

### ⚙️ Backend (Planned / Upcoming)
- **Node.js + Express** — To manage APIs, quizzes, and user submissions.
- **MongoDB (Mongoose)** — For storing user progress, notes, and data.
- **JWT Authentication** — For secure user sessions.
- **RESTful APIs** — To connect frontend with backend efficiently.
